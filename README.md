# Flagged-and-Fragged
 Flagged & Fragged is an intense, 2D platformer shooter built for classic couch rivalry. Grab a friend (or an enemy) and prepare for a fast-paced showdown where being a crack shot is only half the battle. In this arena, the "Flagged" zone is the only place that matters, and your opponent is sitting right next to you, ready to blast you out of it.
